> Философия, специфика и "закиды" по проекту. Живой документ.

---

## Философия продукта

### Главная идея
SaaS-приложение с AI-функциональностью и Telegram ботом.

### Для кого
- Пользователи, которым нужен AI-ассистент
- Доступ через web и Telegram

### Ключевая ценность
- Единый интерфейс для AI
- Система кредитов для монетизации
- Multi-platform (web + bot)

---

## Специфика проекта

### Стек
- Next.js 14 (App Router)
- Python (Telegram бот)
- Prisma + PostgreSQL
- Clerk Auth
- OpenAI API

### Точки входа
1. **Web UI** — основной интерфейс
2. **Telegram бот** — мобильный доступ
3. **API** — интеграции

### Особенности архитектуры
- Server Actions для бизнес-логики
- Отдельный Python бот (не serverless)
- Общая БД для web и бота
